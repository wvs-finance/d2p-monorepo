## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] InstrumentParams.tsx: fork_params_caption `<p>` inside `<dl>` violated HTML spec**
- **Found during:** Task 1 (axe a11y run on simulated route)
- **Issue:** `<div><p>{labels.fork_params_caption}</p></div>` rendered inside `<dl>` triggered axe `definition-list` (only-dlitems) serious violation. HTML spec: `<dl>` children must be `<dt>`, `<dd>`, `<div>`, `<script>`, or `<template>`. A `<div>` containing `<p>` is non-conforming.
- **Fix:** Wrapped the simulated branch return in a React fragment `<>...</>`, moved the caption `<p>` outside the `<dl>` (before it, with `mb-2` spacing). The `<dl>` now contains only conforming `<div>` wrappers for dt/dd pairs.
- **Files modified:** `components/defi/InstrumentParams.tsx`
- **Commit:** `6233fb7`

**2. [Rule 1 - Bug] Playwright CSS attribute selector is case-sensitive for `aria-label*="schematic"`**
- **Found during:** Task 1 (e2e test for en locale payoff aria-label)
- **Issue:** en aria-label is "Schematic payoff diagram — positive gamma" (capital S). CSS `[aria-label*="schematic"]` does not match. Three tests needed fix (instrument-page.spec.ts + defi-instruments.spec.ts).
- **Fix:** Replaced CSS attribute selector with `page.locator('[role="img"][aria-label]')` + `.count()` loop + `/schematic/i` regex check.
- **Files modified:** `tests/e2e/instrument-page.spec.ts`, `tests/a11y/defi-instruments.spec.ts`
- **Commit:** `6233fb7`

**3. [Rule 1 - Bug] Playwright strict mode violation on multiple matching `dd` and `section` elements**
- **Found during:** Task 1 (e2e tests for seededLiquidity and CashFlowWaterfall section)
- **Issue:** `locator('dd').filter({ hasText: '1000000000000000000000000' })` matched 2 elements (InstrumentParams + SnapshotPoolPanel). `locator('section').filter(...)` matched 2 section elements.
- **Fix:** `.first()` on the dd locator; `getByRole('region', { name: 'Flujo de caja' })` for the cashflow section (uses aria-label as unique discriminator).
- **Files modified:** `tests/e2e/instrument-page.spec.ts`
- **Commit:** `6233fb7`
