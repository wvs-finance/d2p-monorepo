### DEFERRED ITEM 3 — InstrumentJsonLd script console warning

The Evidence Collector (05-04 live verification, line 119 of 05-LIVE-VERIFICATION.md) flagged: "Encountered a script tag while rendering React component" when InstrumentJsonLd was in the client tree. This is a React dev-mode warning, not a production error. The component is correctly an RSC with no 'use client' directive. The warning is structural — it occurs when an RSC emits a `<script>` in a tree that also contains client components. No one-line fix exists that doesn't require restructuring the page layout or extracting InstrumentJsonLd to a separate RSC subtree. Deferred to Wave-4 gate for consideration.
