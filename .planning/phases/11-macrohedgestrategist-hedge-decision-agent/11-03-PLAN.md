<objective>
Stand up the Somnia-testnet integration runner `contracts/script/macro-hedge-strategist-e2e.sh` (modeled on the proven `somnia-probe-e2e.sh`) that ensures a deployed `MacroOracle`, deploys `MacroHedgeStrategist`, sequences the keeper steps (refresh `MacroOracle` -> fire `requestActionDecision` capturing the returned `decisionId` -> await the action callback -> fire `requestSizeDecision(decisionId)`), and polls on-chain for `HedgeDecisionMade`.

It budgets the THREE legs by their CLAUDE.md price classes — the oracle refresh is **json-fetch** (0.03 SOMI → `JSON_DEPOSIT = FLOOR + 0.09 STT`), the two infer legs are **llm-inference** (0.07 SOMI → `LLM_DEPOSIT = FLOOR + 0.21 STT` each) — never floor-only (the known `TimedOut` regression). It runs a STANDALONE `inferString` probe FIRST so a `TimedOut` there unambiguously means "wrong LLM_AGENT_ID" (distinct from the cross-leg join failure mode), and proves the live AGENT-03 criteria: in-enum action, in-range size, and decision-moves-with-consensus across two runs with different `consensus`.

This plan is the LIVE half of AGENT-03 (the unit half — auth/replay/clamp/cross-block-join — is Plan 02). It is `autonomous: false`: it spends real STT and has a human-verify checkpoint capturing the `HedgeDecisionMade` tx for the demo video.

Purpose: AGENT-03 live proof + the Agentathon demo-video evidence.
Output: An executable e2e script and a successful manual run showing two distinct on-chain `HedgeDecisionMade` events.
</objective>
