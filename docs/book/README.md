# DS2P Labs — Abrigo Module Docs

This GitBook documents the Abrigo module: a convex hedging instrument suite built on Panoptic V2, targeting Colombian peso (cCOP) FX exposure.

The module is currently in a fork-only / simulated phase. No instruments are deployed on mainnet. All pool state shown in the UI is sourced from fork-fixture constants captured from abrigo-somnia test files, not from live on-chain reads.

## Structure

- **Apps / Abrigo / Instruments** — per-instrument documentation pages, covering provenance tiers, schematic payoff shapes, and cash-flow arithmetic.

Internal research artifacts (`docs/superpowers/`) are not part of this GitBook; they live in the repository for developer reference only.
