**Fork A — Prompt provenance: CALLER-SUPPLIED FREE-TEXT INTENT (4B), chosen over the deterministic macro-factor prompt (4A).**
Rationale (RESEARCH Pattern 4 + Open Q1): 4B matches the UI free-text box, makes the live "different prompt -> different mandate" proof LITERAL and compelling for the "Autonomous performance" judging, and the school is genuinely prompt-derived. Determinism is preserved by the contract-constant `SYSTEM_PROMPT` + `chainOfThought=false` + the registry `allowedValues` (the model cannot return out-of-set); consensus is per fixed (input, seed). The notional leg rebuilds the prompt deterministically from the stored `(userIntent, macroValue, consensus)` + the chosen school's `promptBias` fragment.
