<objective>
Design `impl_history.parquet` (IMPL-01 / KPD-07): track proxy implementation transitions via `Upgraded(address)` ONLY, with a guaranteed floor row `[deployment_block, inf)`, a head-row bytecode-hash backstop that hashes the IMPL (not the proxy stub), and the PITFALLS A1 +/-10-block transition quarantine. The Upgraded-only listener is a deliberate user scoping that SUPERSEDES PITFALLS A2; the bytecode backstop is the load-bearing defense that makes it safe.
